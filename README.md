# Titanic Survival Prediction 🚢

## 📌 Overview
This project aims to predict whether a passenger survived the Titanic disaster using machine learning techniques.

## 📁 Project Structure
- data/ → dataset files
- notebook/ → Jupyter notebooks for analysis and experiments
- src/ → Python scripts for clean implementation

## 🛠 Tools & Technologies
- Python
- Pandas, NumPy, Matplotlib
- Scikit-learn
